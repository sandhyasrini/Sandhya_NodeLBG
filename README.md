<h1> LBG NodeJS Assessment </h1>

### Steps to run

1. `npm install`
2. `npm start`


### API Reference
|Route         | HTTP Verb   |Description         |
|--------------|:-----------:|:------------------:|
|/api/bears    |GET          |Get all the bears   |
|/api/bears    |POST         |Create a new bear   |
|/api/bears/:id|PUT          |Modify a bear       |
|/api/bears/:id|DELETE       |Delete a bear       |
